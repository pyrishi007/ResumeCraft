# ResumeCraft 🧾✨

**ResumeCraft** is a modern frontend web application that helps users create clean, professional resumes quickly with customizable sections and real-time preview. It provides an intuitive interface for building job-ready resumes without requiring design experience.


### Color Scheme for logo
#1F6FEB — Electric Blue (main fill, bold "C" text)
#4DAAFF — Sky Blue (the dot accent / lighter highlight)
#0f0f0f — Near Black (the "r" text)
#FFFFFF — White (stroke/outline on the back square, if on dark bg)
